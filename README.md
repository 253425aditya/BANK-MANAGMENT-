# BANK-MANAGMENT-
Bank Management System (Python)

This is a simple Python project that allows users to manage basic banking operations.
All data is saved in a JSON file (data.json) so the account information stays stored even after the program closes.

Features

Create a new bank account

Deposit money

Withdraw money

View account details

Update account information

Delete an account

Automatically updates and saves data in data.json

How It Works

When the program starts, it loads existing data from data.json

A menu appears where you can choose what you want to do

Each operation asks for your account number and PIN for security

New accounts get a randomly generated account number

Updated data is saved automatically

How to Run

Install Python 3 on your computer

Keep your Python file and data.json in the same folder

Run the script:

python main.py


Choose an option from the menu

Requirements

Python 3

No external libraries needed

Files

main.py → Contains the Bank class and menu

data.json → Stores all account details

Notes

PIN must be 4 digits

You must be 18+ to create an account

You can deposit up to ₹10,000 at once

Age, account number, and balance cannot be changed in the update section
